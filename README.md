# Toppit
## Images
https://imgur.com/a/m7UY8

## Key Features
  - Upvote/Downvote a topic
  - Add comments to a topic
  - Share topics with other uses
  - Create topics arounds themes
  - Crate subtopic
  - Chat with other users in real-time
